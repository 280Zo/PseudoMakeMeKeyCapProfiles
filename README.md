# Pseudo Make Me Key Cap Profile

## How to
* make sure OpenSCAD is up to date.
* unzip libraries.zip into your OpenSCAD libraries
* select keycap you want to render by changing KeyID value
* press F5 to review
* press F6 to render
* press F7 to export as STL
